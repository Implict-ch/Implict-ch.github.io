# 思路1:从递归 -> 记忆化搜索

一般的套路：考虑最后一个位置放或不放物品，

1.放置了，那么第n−1n-1n−1位就不能放，而n−2n-2n−2不受限制，则这种情况方案数为:f(n−2)f(n-2)f(n−2)

2.没放置,那么n−1n-1n−1位不受限制，则这种情况方案数为:f(n−1)f(n-1)f(n−1)

那么递归式:f(n)=f(n−1)+f(n−2)f(n) = f(n-1) + f(n-2)f(n)=f(n−1)+f(n−2)。
