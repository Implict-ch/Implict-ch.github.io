# Q&A

# Q1.直接刷笔试题有用吗？

1.本站不仅提供思路，还针对LeetCode+CodeFun2000做了类似题目推荐。保证大家刷一道题，通一类题。塔子哥一直认为自顶向下的学习法才是最高效的 。塔子哥见过无数群友力扣都刷了400~500题了，到了笔试阶段，表现得还是稀烂！有位群友leetcode都刷了900题了，华为机试还是没到150。这不是在说刷力扣题没用。力扣题针对面试非常有用，因为面试官大部分都从里面抽原题。但是去年秋招和今年春招的笔试来看，有相当一部分笔试出题人不按LeetCode出牌！

2.一般的大厂的出题人是比较稳定的，那么就导致各个厂出题侧重点比较稳定。而出题人的出题偏好，你不看一些真题完全猜不透摸不到(除非你打过oi/acm,那就没所谓谁出了，反正是降维打击咯)。举个例子，某大厂出题人在塔子哥群说过，他们很少根据力扣出题。基本是牛客小白月赛以及codeforces 1500以下的题。所以如果大家有一年以上，每天4个小时的时间专注刷算法题。可以去海量的刷这些竞赛题。(那和直接打竞赛也没啥区别了)
