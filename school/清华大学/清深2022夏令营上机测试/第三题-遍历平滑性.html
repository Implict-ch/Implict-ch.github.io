# 题目内容

给定一棵有 nnn 个节点二叉树 TTT ，树上的每个点 iii 有点权 aia_ia​i​​ 。

考虑二叉树下的先序遍历 t = v_1,…,v_n ，我们定义其平滑性为

D(t)=∑1≤i<n∣avi−avi+1∣D(t)=\sum_{1\le i\lt n} |a_{v_i}-a_{v_{i+1}}|D(t)=∑​1≤i<n​​∣a​v​i​​​​−a​v​i+1​​​​∣

现在，你可以交换任意节点的左右子树，使得树 TTT 的先序遍历的平滑性最小。

# 输入描述

输入的第一行包含一个正整数 n(n≤2000)n(n \leq 2000)n(n≤2000) 。

输入的第二行包含 nnn 个正整数，第 iii 个正整数 aia_ia​i​​ 为节点 iii 的点权，保证 1≤ai≤1091\le a_i \le 10^91≤a​i​​≤10​9​​ 。

输入的第三行包含 n−1n-1n−1 个正整数，第 iii 个正整数 fif_if​i​​ 为节点 i+1i+1i+1 的父节点，保证 1≤fi≤i1 \le f_i\le i1≤f​i​​≤i 且每个 fif_if​i​​ 不会出现超过 222 次。

我们约定 fif_if​i​​ 第一次出现时，表示 i+1i+1i+1 是 fif_if​i​​ 的左儿子， fif_if​i​​ 第二次出现时，表示 i+1i+1i+1 是 fif_if​i​​ 的右儿子。

# 输出描述

输出一个非负整数，即所求的最小平滑性。

# 样例

输入

5
1 2 3 4 5

1 1 2 2

输出

6

# 题解

假设我们在节点uuu , 左右节点分别为l,rl,rl,r . 如果先访问lll 这一边，那么在lll 中最后一个访问的节点，它需要和rrr 的值作绝对值差。之后访问rrr 时，rrr 中最后一个访问的节点，它需要和之前的某个节点做差。

然后又观察到nnn 比较小。我们自然想到动态规划,一个维度存当前点，另一个维度存最后一个点要去和哪个点做差:

dpu,gdp_{u,g}dp​u,g​​ 代表以uuu 点为子树，且本子树最后访问的那个节点需要和ggg 点作差的最小遍历平滑性。

转移为:

dpu,g=min(∣au−al∣+dpl,r+dpr,g,∣au−ar∣+dpr,l+dpl,g)\large dp_{u,g} = min(|a_u-a_l| + dp_{l,r} + dp_{r,g},|a_u-a_r| + dp_{r , l} + dp_{l,g}) dp​u,g​​=min(∣a​u​​−a​l​​∣+dp​l,r​​+dp​r,g​​,∣a​u​​−a​r​​∣+dp​r,l​​+dp​l,g​​)

记忆化搜索即可.O(n2)O(n^2)O(n​2​​)

#include <bits/stdc++.h>
using namespace std;
#define ll long long
const int maxn  = 5005;
int a[maxn];
int t[maxn][2];
int dp[maxn][maxn];
int dfs (int u , int x){
    int l = t[u][0];
    int r = t[u][1];
    if (!l) {
        if (x == 0) return 0;
        return abs(a[u] - a[x]);
    }
    if (~dp[u][x]) return dp[u][x];
    if (!r){
        return dp[u][x] = abs(a[u] - a[l]) + dfs(l , x);
    }
    int ans1 = abs(a[u] - a[l]) + dfs(l , r) + dfs(r , x);
    swap(l , r);
    int ans2 = abs(a[u] - a[l]) + dfs(l , r) + dfs(r , x);
    return dp[u][x] = min(ans1 , ans2);
}
int main() {
    int n;
    cin >> n;
    for (int i = 1 ; i <= n ; i++) cin >> a[i];
    for (int i = 2 ; i <= n ; i++){
        int rt;
        cin >> rt;
        if (t[rt][0]) t[rt][1] = i;
        else t[rt][0] = i;
    }
    memset(dp , -1 , sizeof dp);
    cout << dfs(1 , 0) << endl;
    return 0;
}
