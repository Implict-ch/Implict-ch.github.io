# 题目描述

塔子哥有一个 n×mn\times mn×m 的矩阵，他想问你这个矩阵中有多少个元素的个位为 999 。

# 输入描述

第一行，一个整数 n(1≤n,m≤100)n(1 \leq n,m \leq 100)n(1≤n,m≤100)，表示数组的行数和列数。
接下来 nnn 行，每行 mmm 个整数，第 iii 行 第 jjj 个整数为 ai,j(1≤ai,j≤109)a_{i,j}(1 \leq a_{i,j} \leq 10^9)a​i,j​​(1≤a​i,j​​≤10​9​​) 。

# 输出描述

一个非负整数，表示这个矩阵中个位为 999 的元素数。

# 样例

输入

3 3
99 89 29
22 44 43
55 33 11

输出

3

# 思路：模拟

遍历矩阵，模10判断结果是否为9即可。

时间复杂度：O(nm)O(nm)O(nm)

# 代码

n, m = map(int, input().split())
a = [list(map(int, input().split())) for _ in range(n)]

ans = 0
i = j = 0
for i in range(n):
    for j in range(m):
        if a[i][j] % 10 == 9:
            ans += 1

print(ans)
