# 题目内容

塔子哥是一个游戏迷，最近沉迷于一款飞机大战游戏中。在这个游戏中，他需要控制一架飞机发射子弹，来消灭出现在屏幕中的敌人。为了让游戏更加有趣和挑战性，游戏设置了两种不同的发射按键，每个按键都有不同的特殊效果。

在游戏中，按下 AAA 键，飞机会发射出 222 枚子弹，每个子弹会对命中的敌人造成 111 点固定伤害，但不能作用于同一个敌人。而按下 BBB 键，飞机会发射出 111 枚子弹，该子弹能够对命中的敌人造成巨额伤害并瞬间将其秒杀。

整个游戏一共有 TTT 个关卡，每个关卡会在屏幕中同时出现 NNN 个敌人，这 NNN 个敌人所能承受的伤害也已经确定。当塔子哥消灭当前关卡的所有敌人后，发射出去多余的子弹会消失，游戏会自动进入下一个关卡。

作为一名游戏高手，塔子哥总是能够操纵子弹命中想要攻击的敌人。但是，他还是很好奇每个关卡自己最少需要按几次发射按键才能将所有敌人全部消灭。他找到了你，希望你能够帮助他解决这个问题。

# 输入描述

第一行输入一个固定数字 TTT （ 1≤T≤10001\le T \le 10001≤T≤1000 ）表示关卡的总数量， NNN （ 1≤N≤2001\le N\le 2001≤N≤200 ）表示每个关卡出现的敌人数量。

接下来 TTT 行，每行有 NNN 个数字 D_1，D_2，.....,D_N （ 1≤Di≤2001\le D_i \le 2001≤D​i​​≤200 ）分别表示这 NNN 个敌人所能承受的伤害。

# 输出描述

结果共有 NNN 行，每行一个数字，分别表示对于这个关卡，最少按几次发射按键就可以将敌人全部消灭。

# 样例

输入

3 3
1 2 1
2 3 2
1 2 3

输出

2

3

3

样例解释

游戏共有 333 个关卡，每个关卡会出现 333 个敌人。

第一个关卡，先按下 AAA 键控制子弹消灭第 111 个和第 333 个敌人后，再按下 BBB 键消灭第二个敌人。所以最少按 222 次。

第二个关卡，按下 333 次 BBB 键分别消灭这 333 个敌人。

第三个关卡，按下 333 次 BBB 键分别消灭这 333 个敌人。也可以按 333 次 AAA 键，敌人剩余承受伤害的变化为： [1, 2, 3] ->[1，1，2]->[1,0,1]->[0，0，0]

# 思路：贪心

第 iii 个敌人所能承受的伤害为 hih_ih​i​​

当 hi=1h_i = 1h​i​​=1，则按 111 次 AAA 即可。假设有 oneoneone 个 hi=1h_i=1h​i​​=1，则按 (one+1)/2(one + 1) / 2(one+1)/2 次即可消灭所有 hi=1h_i=1h​i​​=1 的敌人

(one+1)/2(one + 1) / 2(one+1)/2 表示 one/2one / 2one/2 上取整。

当 hi>1h_i > 1h​i​​>1，

  • 如果按 BBB 键，只需要按 kkk 次

  • 如果按 AAA 键，假设有 kkk 个 hi>1h_i>1h​i​​>1 的敌人。这里先考虑kkk 个 hi=2h_i=2h​i​​=2 的情况。

    • 当 kkk 为偶数，则每按两次 AAA 键，可以消灭两个敌人，一共需要按 k2\frac{k}{2}​2​​k​​ 回两次，即 k2×2=k\frac{k}{2}\times 2=k​2​​k​​×2=k 次。
    • 当 kkk 为奇数，对于 k−1k-1k−1 来说，等于上面的偶数情况，即 k−1k-1k−1 次，多出来的一人，额外需要两次，故总共是 k−1+2=k+1k-1+2=k+1k−1+2=k+1 次。

    这样来看，按 BBB 键消灭这些敌人的按键数一定不会多于按 AAA 键消灭这些敌人的按键数。

    这是在 hi>1h_i>1h​i​​>1 的时候，所有的 hi=2h_i=2h​i​​=2 的情况，实际上，当 hi>2h_i>2h​i​​>2 的时候，需要的按键数一定不会比 hi=2h_i=2h​i​​=2 的时候变少。

所以对于 hi>1h_i>1h​i​​>1 的情况，全部选择按 BBB 键。

# C++

#include <bits/stdc++.h>
using namespace std;

const int N = 210;
int d[N];
int T, n;

int main()
{
    scanf("%d%d", &T, &n);
    for (int i = 1; i <= T; ++i) {
        int one = 0;
        for (int j = 1; j <= n; ++j) {
            scanf("%d", &d[j]);
            // 统计 1 的个数
            one += d[j] == 1;
        }

        // = 1 选择 A 键
        int eq1 = (one + 1) / 2;
        
        // > 1 选择 B 键
        int gt1 = n - one;

        printf("%d\n", eq1 + gt1);
    }


    return 0;
}

        }
    }
}
