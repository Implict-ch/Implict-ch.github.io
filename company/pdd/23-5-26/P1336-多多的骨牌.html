# 题目内容

塔子哥穿越到战国时期，成为了一名将军。这天塔子哥在操练士兵，塔子哥在一条直线上安排了nnn名士兵，假设这些士兵只考虑高度，不考虑宽度，第iii名士兵在位置xix_ix​i​​，其高度为hih_ih​i​​，塔子哥可以任意选择一些士兵，命令其按顺序向左或者向右卧倒，向左卧倒之后会占据[xix_ix​i​​-hih_ih​i​​,xix_ix​i​​]个位置，向右卧倒后会占据[xix_ix​i​​,xix_ix​i​​+hih_ih​i​​]位置。要求卧倒的士兵不能触碰到其他之前已经卧倒的士兵或者站立的士兵。问塔子哥最多可以命令多少个士兵卧倒。

# 输入描述

输入第一行为一个数字nnn（111≤\leq≤nnn≤\leq≤10510^{5}10​5​​)，表示有nnn个士兵。

接下来nnn行，每行输入两个数字xix_ix​i​​和hih_ih​i​​（000≤\leq≤xix_ix​i​​≤\leq≤10910^{9}10​9​​,111≤\leq≤hih_ih​i​​≤\leq≤10910^{9}10​9​​),表示第iii个士兵的位置和高度。

输入保证xix_ix​i​​递增且不会出现相等的情况。

# 输出描述

输出一个整数ansansans，表示最多能卧倒的士兵数。

# 样例一

输入

3
1 1
2 1
3 1

输出

2

说明

先让第一个士兵往左卧倒。再让第三个士兵往右卧倒。第二个士兵无论往左还是往右，都会触碰到其他士兵。

# 贪心

一句话贪心策略：每个人先考虑其是否可以向左卧倒，不可以向左卧倒的情况下，再考虑是否可以向右卧倒。

接下来是贪心的 证明 第111个人向左，第nnn个人向右。这样第111个人右边的空间和第nnn个人左边的空间就不会被这两个人占用。

接下来我们考虑第222个人到第n−1n-1n−1个人这n−2n-2n−2个人的卧倒情况。

对于第i(i∈[2,n−1])i(i\in [2, n-1])i(i∈[2,n−1])个人，因为考虑到第iii个人的时候，前i−1i-1i−1个人都已经考虑过了。 故考虑第iii个人

  • 如果其左边还有空间使得其向左卧倒，因为其左边空间已经不会被其他人用了，所以直接卧倒即可。

  • 如果其左边空间不够其向左卧倒，考虑向右卧倒的情况

    • 如果其右边空间不够其向右卧倒，则说明第iii个人无法卧倒
    • 如果其右边空间足够其卧倒，则第iii个人向右卧倒即可。

    你可能会有疑问，如果让第iii个人向右卧倒，是否会影响后面的人的卧倒情况？

    但是我们这里是要最大化卧倒人数的数量，所以我们这么考虑：

    对于第iii个人，如果其右边空间足够其向右卧倒，那么第iii个人向右卧倒的情况下可以卧倒的人数，一定不会比第iii个人不向右卧倒的情况下可以卧倒的人数更少。

    假设第iii个人向右卧倒后，第i+1i+1i+1个人就不可以向左卧倒（即情况为第iii个人不向右卧倒，则第i+1i+1i+1个人是可以向左卧倒的）了，那是不是说明，如果第iii个人不向右卧倒，则第i+1i+1i+1个人就可以向左卧倒，这就相当于，这个空间内要么让第iii个人卧倒，要么让第i+1i+1i+1个人卧倒。

    情况一：选择让第iii个人向右卧倒，第i+1i+1i+1个人还可以继续考虑向右卧倒，注意第nnn个人是可以无条件向右卧倒的。

    情况二：不让第iii个人向右卧倒，那么让第i+1i+1i+1个人向左卧倒，后续的第i+2i+2i+2个人到第nnn个人的卧倒数定义为f(i+2)f(i+2)f(i+2)。

    这两种情况下，如果在情况一，我们先不让第i+1i+1i+1个人向右卧倒了，那么其之后的第i+2i+2i+2个人到第nnn个人的卧倒数也为f(i+2)f(i+2)f(i+2)，所以在这两种情况下进行计算第iii个人到第nnn个人的卧倒数都为1+f(i+2)1+f(i+2)1+f(i+2)。

    注意，在情况一，后续的第i+2i+2i+2个人到第nnn个人的卧倒情况都确定了，第i+1i+1i+1个人仍然可以向右卧倒，这种情况存在的。所以情况一的卧倒人数一定不会比情况二少。

    所以我们采取先考虑向左卧倒，左边无法卧倒后再考虑向右卧倒的贪心策略。

# C++

// 动态规划
#include <bits/stdc++.h>
using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> x(n + 1), h(n + 1);
    for (int i = 1; i <= n; ++i) {
        cin >> x[i] >> h[i];
    }

    // dp[i][0] 表示向左，dp[i][1] 表示不动，dp[i][2] 表示向右
    vector<vector<int>> dp(n + 1, vector<int>(3, 0));
    dp[1][0] = 1;
    dp[1][1] = 0;
    if (n == 1 || x[1] + h[1] < x[2]) dp[1][1] = 1;

    for (int i = 2; i <= n; ++i) {
        // 第 i 个向左
        if (x[i] - h[i] > x[i - 1]) {
            // 第 i-1 个不动
            dp[i][0] = dp[i - 1][1] + 1;
            // 第 i-1 个向左
            if (i - 1 == 1 || x[i - 1] - h[i - 1] > x[i - 2]) dp[i][0] = max(dp[i][0], dp[i - 1][0] + 1);
            // 第 i-1 个向右
            if (x[i - 1] + h[i - 1] < x[i] - h[i]) dp[i][0] = max(dp[i][0], dp[i - 1][2] + 1);
        }

        // 第 i 个不动

        // 第 i-1 个不动
        dp[i][1] = dp[i - 1][1];
        // 第 i-1 个向左
        if (i - 1 == 1 || x[i - 1] - h[i - 1] > x[i - 2]) dp[i][1] = max(dp[i][1], dp[i - 1][0]);
        // 第 i-1 个向右
        if (x[i - 1] + h[i - 1] < x[i]) dp[i][1] = max(dp[i][1], dp[i - 1][2]);

        // 第 i 个向右
        if (i == n || x[i] + h[i] < x[i + 1]) {
            // 第 i-1 个不动
            dp[i][2] = dp[i - 1][1] + 1;
            // 第 i-1 个向左
            if (i - 1 == 1 || x[i - 1] - h[i - 1] > x[i - 2]) dp[i][2] = max(dp[i][2], dp[i - 1][0] + 1);
            // 第 i-1 个向右
            if (x[i - 1] + h[i - 1] < x[i]) dp[i][2] = max(dp[i][2], dp[i - 1][2] + 1);
        }
    }

    printf("%d\n", max({dp[n][0], dp[n][1], dp[n][2]}));

    return 0;
}

// 贪心
#include <bits/stdc++.h>
using namespace std;

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> x(n), h(n);
    for (int i = 0; i < n; ++i) {
        cin >> x[i] >> h[i];
    }

    int ans = 1;
    int R = x[0];
    for (int i = 1; i < n; ++i) {
        // 先考虑向左
        if (x[i] - h[i] > R) {
            R = x[i];
            ans += 1;
        } else {
            // 考虑向右
            if (i + 1 == n || x[i] + h[i] < x[i + 1]) {
                R = x[i] + h[i];
                ans += 1;
            } else {
                // 无法向右和向左，则保持不动
                R = x[i];
            }
        }
    }

    cout << ans << "\n";

    return 0;
}
