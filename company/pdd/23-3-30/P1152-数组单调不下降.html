# 题目内容

曾经有一个塔子哥，他是一个热爱数学的年轻人，他喜欢思考数学问题，研究各种算法。

有一天，他遇到了一个非常有趣的问题。他的朋友给了他一个长度为 nnn 的数组 aaa，让他通过一系列的操作使得数组单调不下降，也就是说，对于任意的 iii 和 jjj，如果 i≤ji \le ji≤j，那么 ai≤aja_i \le a_ja​i​​≤a​j​​。

在听到这个问题的时候，塔子哥很兴奋，因为他知道这个问题可以用他最擅长的算法来解决。他想到了一种方法，可以通过将数组中某些数变成 000 的方式来达到目标。具体地说，他可以选择数组中的一个数 xxx，然后将所有等于 xxx 的数都变成 000。这样做可以让某些数“消失”，也就是让它们不再影响数组单调不下降的性质。但是，这个方法有一个缺点，就是它需要重复执行很多次，才能让整个数组单调不下降。

塔子哥觉得这个问题非常有趣，于是决定尝试解决它。他开始思考如何通过最少的操作次数，使得整个数组单调不下降。

有 ttt 组输入。

# 输入描述

输入第一行为一个整数 ttt ，代表 ttt 组输入。（ 1≤t≤1001\le t\le 1001≤t≤100 ）

每一组输入，第一行为一个整数 nnn 。

第二行为 nnn 个整数。

（ 1≤n,ai≤1041\le n,a_i\le 10^41≤n,a​i​​≤10​4​​ ）

# 输出描述

输出最小的操作次数。

# 样例

输入

3
3
5 1 5
5
1 1 2 2 2
3
1 4 1

输出

2
0
2

# 思路：思维+哈希表

根据题目描述，要求将原数组转换为一个单调不下降的数组。

如果原数组不是一个单调不下降的数组，则必然存在 i<ni<ni<n，有 ai>ai+1a_i>a_{i+1}a​i​​>a​i+1​​。

找到最大的索引 j<nj<nj<n，满足 aj>aj+1a_{j}>a_{j+1}a​j​​>a​j+1​​，且 aj+1≤aj+2≤⋯≤an−1≤ana_{j+1}\leq a_{j+2}\leq \cdots \leq a_{n-1}\leq a_na​j+1​​≤a​j+2​​≤⋯≤a​n−1​​≤a​n​​。

此时 i∈[1,j]i\in [1,j]i∈[1,j] 的 aia_ia​i​​ 都需要调整为 000。

此外，对于 k∈[j+1,n]k\in [j+1,n]k∈[j+1,n] 的 aka_ka​k​​ 与 j∈[1,j]j\in [1,j]j∈[1,j] 的 aja_ja​j​​ 中的任意一个值相同的情况下，aka_ka​k​​ 也需要调整为 000，相应的，也需要找到这部分中最大的索引，使得这个索引和其之前的值都调整为 000。

这里找到 i∈[1,j]i\in [1,j]i∈[1,j] 中的每个 aia_ia​i​​ 的最靠后位置可以用一个哈希表实现，因为值域为 100001000010000，故可以用一个数组作为哈希表。

时间复杂度：O(n)O(n)O(n)

# CPP

#include <bits/stdc++.h>
using namespace std;

void solve() {
    int n;
    cin >> n;

    vector<int> a(n);

    // 每个值最靠后的位置
    vector<int> right(10001, -1);

    for (int i = 0; i < n; ++i) {
        cin >> a[i];
        // 因为是从前往后遍历，所以越往后遍历到的一定是越靠后的位置
        right[a[i]] = i;
    }

    int ans = 0;
    int now = n - 2;
    while (now >= 0) {
        // 找到最远的索引 now 满足 a[now] > a[now + 1]
        if (a[now] > a[now + 1]) {
            break;
        } else {
            now -= 1;
        }
    }

    // 找到从 a[0] 到 a[now] 这些值中最远的索引 R
    // a[0] 到 a[R] 都被赋值为 0
    int R = now;
    for (int i = now; i >= 0; --i) {
        R = max(R, right[a[i]]);
    }

    // 值相同的索引只需要执行一次即可将这些索引对应的值都变成 0
    // 可以用每个值最靠后的索引来作为使用标记，防止相同的值重复累加答案
    for (int i = R; i >= 0; --i) {
        if (a[i] && i == right[a[i]]) ans += 1;
    }

    cout << ans << "\n";
}

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) solve();

    return 0;
}

        # 因为是从前往后遍历，所以越往后遍历到的一定是越靠后的位置
        right[a[i]] = i

    ans = 0
    now = n - 2
    while now >= 0:
        # 找到最远的索引 now 满足 a[now] > a[now + 1]
        if a[now] > a[now + 1]:
            break
        else:
            now -= 1

    # 找到从 a[0] 到 a[now] 这些值中最远的索引 R
    # a[0] 到 a[R] 都被赋值为 0
    R = now
    for i in range(now, -1, -1):
        R = max(R, right[a[i]])

    # 值相同的索引只需要执行一次即可将这些索引对应的值都变成 0
    # 可以用每个值最靠后的索引来作为使用标记，防止相同的值重复累加答案

    a := make([]int, n)

    // 每个值最靠后的位置
    right := make([]int, 10001)
    for i := 0; i < 10001; i++ {
        right[i] = -1
    }

        scanner.Scan()

            now -= 1
        }

    // 找到从 a[0] 到 a[now] 这些值中最远的索引 R

    R := now

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
