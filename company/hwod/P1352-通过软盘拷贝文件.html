# 题目描述

有一名科学家想要从一台古董电脑中拷贝文件到自己的电脑中加以研究

但此电脑除了有一个3.53.53.5寸软盘驱动器以外，没有任何手段可以将文件持贝出来，而且只有一张软盘可以使用。

因此这一张软盘是唯一可以用来拷贝文件的载体。

科学家想要尽可能多地将计算机中的信息拷贝到软盘中，做到软盘中文件内容总大小最大。已知该软盘容量为147456014745601474560字节。文件占用的软盘空间都是按块分配的，每个块大小为512512512个字节。

一个块只能被一个文件使用。拷贝到软盘中的文件必须是完整的，且不能采取任何压缩技术。

# 输入描述

第111行为一个整数NNN，表示计算机中的文件数量。1≤N≤10001\leq N \leq 10001≤N≤1000

接下来的第222行到第N+1N+1N+1行(共NNN行)，每行为一个整数，表示每个文件的大小SiSiSi，单位为字节。

O≤i≤NO \leq i \leq NO≤i≤N,0≤Si0 \leq Si0≤Si

# 输出描述

科学家最多能拷贝的文件总大小

备注

为了充分利用软盘空间，将每个文件在软盘上占用的块记录到本子上。即真正占用软盘空间的只有文件内容本身。

# 样例

输入

3
737270
737272
737288

输出

1474542

说明

333个文件中，每个文件实际占用的大小分别为737280737280737280字节、737280737280737280字节、737792737792737792字节.只能选取前两个文件，总大小为147454214745421474542字节。

虽然后两个文件总大小更大且未超过147456014745601474560字节，但因为实际占用的大小超过了147456014745601474560字节，所以不能选后两个文件。

输入

6
400000
200000
200000
200000
400000
400000

输出

1400000

说明

从666个文件中，选择333个大小为400000400000400000的文件和111个大小为200000200000200000的文件，得到最大总大小为140000014000001400000。

也可以选择222个大小为400000400000400000的文件和333个大小为200000200000200000的文件，得到的总大小也是140000014000001400000。

# 思路：01背包

01背包模板题。考虑软盘共有1474560512=2880\frac{1474560}{512}=2880​512​​1474560​​=2880个块。 而每个块只能被一个文件占用，这意味着当存在一个块的512字节未被用满时，其未被使用的字节也不能被其他文件使用。

题目要求的是，在288028802880个块的条件下，选择一些文件，使得这些文件可以被288028802880块装填，且这些文件的字节数之和最大。

那么将288028802880个块看作背包容量，每个文件需要占用的块的数量为物品体积，每个文件的实际字节数为物品价值。

这样就是一个裸的 01 背包问题了。

时间复杂度:O(2880n)O(2880n)O(2880n)

# 代码

# C++

#include <bits/stdc++.h>
using namespace std;

const int V = 2880;

int main()
{
    int n;
    cin >> n;

    // 背包体积为 V
    vector<int> f(V + 1);
    for (int i = 1; i <= n; ++i) {
        // 每个物品的价值为 w
        int w; cin >> w;
        // 每个物品的体积为 v
        int v = (w + 511) / 512;
        for (int j = V; j >= v; --j)
            f[j] = max(f[j], f[j - v] + w);
    }

    printf("%d\n", f[V]);

    return 0;
}

package main

import (
    "fmt"

const V = 2880

func main() {
    var n int
    fmt.Scan(&n)

    // 背包体积为 V
    f := make([]int, V+1)
    for i := 1; i <= n; i++ {
        // 每个物品的价值为 w
        var w int
        fmt.Scan(&w)
        // 每个物品的体积为 v
        v := (w + 511) / 512
        for j := V; j >= v; j-- {
            f[j] = max(f[j], f[j-v]+w)
        }
    }
    fmt.Printf("%d\n", f[V])
}

func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}
