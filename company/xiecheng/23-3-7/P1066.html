# 题目内容

塔子哥是一个热爱数学的高中生。他最近在研究数列的性质，尤其是对于一些特殊的数列，他想知道它们是否稳定。

经过一番思考，他定义了一个数组为“稳定的”，当且仅当数组中相邻的两个元素之差的绝对值不超过 111 。

例如，对于数组 [2,3,2,2,1][2,3,2,2,1][2,3,2,2,1] ，它是稳定的，因为相邻元素之差不超过 111 。

而数组 [1,3,2][1,3,2][1,3,2] 则不是稳定的，因为 111 和 333 之间的差的绝对值超过了 111 。

现在，他拿到了一个由 nnn 个整数组成的数组 aaa ，他想要求出该数组的最长的稳定的连续子数组的长度。他想知道这个长度，以便研究该数组的性质。你能帮帮他吗？

# 输入描述

第一行输入一个正整数 nnn ，代表数组的大小。

第二行输入 nnn 个正整数 aia_ia​i​​ ，代表数组的元素。

1≤n≤1000001 \le n \le 1000001≤n≤100000 ，1≤ai≤1091\le a_i \le 10^91≤a​i​​≤10​9​​

# 输出描述

一个正整数，代表最长连续稳定子数组的长度。

# 样例

输入

6
2 3 5 4 5 6

输出

4

# 思路：贪心

我们需要找到给定数组中最长的稳定连续子数组的长度，即满足相邻元素之差的绝对值不超过 1 的子数组。我们需要子数组尽可能的长，所以只有当相邻两个

元素大于1时我们才考虑将子数组截断，计算子数组的长度，这里需要注意的是可能存在遍历到数组结尾数组依然稳定，所以可以遍历到数组的下一位，统计

该子数组的结果。子数组长度的计算是记录数组的起始点L与结束点的下一位R，所以子数组长度为R-L，截断一次子数组了，下一次子数组的起始点更新为R。

# CPP

#include <bits/stdc++.h>
using namespace std;

int in() {
	int x;
	cin >> x;
	return x;
}
int main()
{
    int n = in();
    int a[n];
    for (int i = 0; i < n; i++) {
        a[i] = in();
    }
    int L = 0, ans = 0;
    for(int R = 1; R <= n ; R++) {
        //Math.abs(a[R] - a[R-1]) > 1说明当前位置不满足条件了，此时维护的区间为[L, R-1]，区间长度就是R-L
        if(R == n || abs(a[R] - a[R-1]) > 1) {
            ans = max(ans, R - L);
            L = R;
        }
    }
    cout << ans << endl;
}

# python

def main():
    n = int(input())
    a = list(map(int,input().split()))
    L = 0
    ans = 0
    for R in range(1, n + 1):
        if R == n or abs(a[R] - a[R-1]) > 1:
            ans = max(ans, R - L)
            L = R
    print(ans)

if __name__ == "__main__":
    main()

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = scanner.nextInt();
        }
        int L = 0, ans = 0;
        for (int R = 1; R <= n; R++) {
            if (R == n || Math.abs(a[R] - a[R - 1]) > 1) {
                ans = Math.max(ans, R - L);
                L = R;
            }
        }
        System.out.println(ans);
    }
}

    const a = lines[1].split(' ').map(Number);
    processInput(n, a);

function processInput(n, a) {
    let L = 0;
    let ans = 0;

    for (let R = 1; R <= n; R++) {
        if (R === n || Math.abs(a[R] - a[R - 1]) > 1) {
            ans = Math.max(ans, R - L);
            L = R;
        }
    }
    console.log(ans);
}
