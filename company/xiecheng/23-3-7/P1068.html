# 题目内容

塔子哥是一位热爱自驾游的旅行者，他的车子非常特别。和普通的汽车不同，他的车子的最高速度和油量成正比，即油越多，最高速度越快，并且，行驶过程中油是不会消耗。

为了让自己的旅行更加顺利，塔子哥决定提前加足油，以便在行驶过程中始终保持最高速度。他的车初始的最高速度为 v0v_0v​0​​，但当他加油 ttt 个时间单位后，车的最高速度会变成 v0+t×xv_0+t \times xv​0​​+t×x。

塔子哥计划开车行驶总里程为 yyy，他希望在保持最高速度的前提下，尽可能地节省时间。假设他不需要考虑加速的时间，你能帮助他计算出他至少需要花费多少时间才能完成旅行吗？

# 输入描述

三个整数 v0v_0v​0​​ 、 xxx 、 yyy ，用空格隔开。

0≤v0≤1090\le v_0 \le 10^90≤v​0​​≤10​9​​

1≤x,y≤1091\le x,y \le 10^91≤x,y≤10​9​​

# 输出描述

一个浮点数，代表最终花费的总时间。

如果你的答案和标准答案的相对误差不超过 10−610^{-6}10​−6​​ ，则认为答案正确。

# 样例

输入

1 3 5

输出

2.24866

# 思路

前置知识：Oi-Wiki 三分 (opens new window)

在这道题中，我们需要找到一个时间点，使得在这个时间点上达到目的地所需的总时间最小。根据问题的描述，车辆的最高速度和油量成正比，即油越多，最

高速度越快，行驶时间越少，但与此同时，加油消耗的时间越多。可以想到，这样的时间点只有一个，而低于和高于这个时间点需要的时间都是递增的。而使

用三分搜索可以解决这个问题。三分搜索适用于在一个单峰函数中寻找极值点的情况，而车辆的总时间随时间变化的情况可以看作是一个单峰函数。在每个时

间点上，我们可以计算到达目的地所需的总时间，并根据总时间的变化趋势确定下一步搜索的范围。

# CPP

#include <bits/stdc++.h>
using namespace std;

int v, x, y;
double res = 1e9;
double calc(double t1) {
    double t = y / (v+t1*x) + t1;
    res = min(res, t);
    return t;
}
int main()
{
	cin >> v >> x >> y;
    double L = 0, R = 1e9;
    for(int i = 0 ; i <= 5000 ; i ++){
        double lmid=L+(R-L)/3;
        double lres = calc(lmid);
        double rmid=R-(R-L)/3;
        double rres = calc(rmid);
        if (lres <= rres) {
            R = rmid;
        } else {
            L = lmid;
        }
    }
    printf("%.5lf\n", res);
}

# python

import sys

v, x, y = map(int, sys.stdin.readline().split())
res = 1e9

def calc(t1):
    global res
    t = y / (v + t1 * x) + t1
    res = min(res, t)
    return t

L, R = 0, 1e9
for _ in range(5000):
    lmid = L + (R - L) / 3
    lres = calc(lmid)
    rmid = R - (R - L) / 3
    rres = calc(rmid)
    if lres <= rres:
        R = rmid
    else:
        L = lmid

print(f"{res:.5f}")
