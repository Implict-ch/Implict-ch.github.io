# 题目内容

塔子哥收到了一封神秘的信件，里面包含了一串长度为 nnn 的字符串。他注意到这个字符串非常特殊，其中每个字母都可以被重复任意次数。他很好奇，如果把这个字符串的某个子区间重复一遍，会不会得到一个更长的字符串呢？

于是塔子哥开始进行操作，他每次会选择一个区间 [l,r][l,r][l,r]，将第 lll 个字母到第 rrr 个字母各重复一次，并插入到该字母的后面。例如，对于字符串 "abcd"，若选择区间 [2,3][2,3][2,3] 进行操作，字符串将变成 "abbccd"。

塔子哥一共进行了 qqq 次操作，他问你在 qqq 次操作结束后，最终的字符串是什么样子？

# 输入描述

第一行输入两个正整数 nnn 和 qqq ，分别代表字符串长度和操作次数。

第二行输入一个仅由小写英文字母组成的字符串，代表初始的字符串。

接下来的 qqq 行，每行输入两个正整数 lll ， rrr ，代表操作的区间。

1≤n≤10001\le n\le 10001≤n≤1000

1≤q≤101\le q\le 101≤q≤10

1≤l≤r≤1061\le l \le r \le 10^61≤l≤r≤10​6​​

保证每次操作时， rrr 不大于当前的字符串长度。

# 输出描述

一个字符串，代表所有操作结束后形成的字符串。

# 样例

输入

5 3
abcde
2 4
5 8
1 3

输出

aabbbbcccddddee

样例解释

第一次操作后，字符串变成 abbccddef

第二次操作后，字符串变成 abbbccccdddef

# 思路:模拟

根据题意，先定义一个新的字符串ttt，用来存储一次询问修改完后的字符串，将给定区间的字符重复两次，其他字符串不做更改。每次询问后将ttt字符串赋值

给sss，重复qqq次询问。

# 代码

# CPP

#include <bits/stdc++.h>
using namespace std;

int main()
{
	int n, q;
	string s;
	cin >> n >> q >> s;
	while(q--) {
		int l, r;
		cin >> l >> r;
		string t;
		for(int i = 0 ; i < s.size() ; i ++) {
			t += s[i];
			if(i >= l-1 && i <= r-1) t += s[i];
		}
		s = t;
	}
	cout << s << endl;
}

# Java

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        int q = scanner.nextInt();
        String s = scanner.next();
        while (q-- > 0) {
            int l = scanner.nextInt();
            int r = scanner.nextInt();
            StringBuilder t = new StringBuilder();
            for (int i = 0; i < s.length(); i++) {
                t.append(s.charAt(i));
                if (i >= l - 1 && i <= r - 1) {
                    t.append(s.charAt(i));
                }
            }
            s = t.toString();
        }
        System.out.println(s);
    }
}
