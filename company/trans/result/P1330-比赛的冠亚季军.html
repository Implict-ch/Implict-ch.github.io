# 题目描述

有N(3≤N<10000)N (3\leq N< 10000)N(3≤N<10000)个运动员，他们的ididid为000到N−1N-1N−1,他们的实力由一组整数表示。他们之间进行比赛，需要决出冠亚军。比赛的规则是000号和111号比赛，222号和333号比赛，以此类推，每一轮，相邻的运动员进行比赛，获胜的进入下一轮，实力值大的获胜，实力值相等的情况，ididid小的情况下获胜，轮空的直接进入下一轮。

# 输入描述

输入一行NNN个数字代表NNN的运动员的实力值(0≤(0\leq(0≤实力值≤10000000000)\leq10000000000)≤10000000000)。

# 输出描述

输出冠亚季军的ididid，用空格隔开。

# 样例

输入

2 3 4 5

输出

3 1 2

说明

第一轮比赛，

ididid为000实力值为222的运动员和ididid为111实力值为333的运动员比赛，111号胜出进入下一轮争夺冠亚军，

ididid为222的运动员和ididid为333的运动员比赛，333号胜出进入下一轮争夺冠亚军，

冠亚军比赛，333号胜111号。

故冠军为333号，亚军为111号，222号与000号，比赛进行季军的争夺，222号实力值为444，000号实力值222，故222号胜出，得季军,冠亚季军为333111222。

# 思路：模拟

现给定N(10000>N≥3)N(10000\gt N\ge3)N(10000>N≥3)个运动员与他们各自的实力值，每轮比赛中，从编号小的运动员开始，两两pk，获胜者进入下一轮，而失败者淘汰。获胜方式为：

  1. 实力值大的获胜
  2. 实力值相等的编号小的获胜
  3. 没pk对象的直接获胜

我们需要求出整场比赛下来的冠亚季军。

看数据规模和题目意思，可以很快发现这是一道模拟题。那么关键就在于怎么模拟了。模拟方法每个人有各自的理解，因此模拟方法也可能不同。这里给出一个模拟方法供参考。

定义:

结构体：

struct node{
	int id;
	ll value;
};

用于存储单个运动员的编号以及其实力值。

数组：win[]win[]win[]，lose[]lose[]lose[]记录胜者组、败者组有哪些运动员

变量：winnumberwinnumberwinnumber，losenumberlosenumberlosenumber分别记录胜者组、败者组有多少人

每次在胜者组中从小到大两两比较运动员，获胜者进入胜者组，而失败者则加入败者组。

注意，如果不使用空间复用的话，可以创建一个临时数组tmptmptmp，获胜者先存储在临时数组tmptmptmp中，待该轮结束后，在转移到数组winwinwin中，从而开始下一轮比拼。

但是我们发现，我们每次比较两个人，只会产生一个胜者，那么winwinwin数组的空间是可以复用的。具体来说，当我们比较完111号以及222号运动员后，将获胜者存储在111号的位置。而比较完333号和444号运动员后，将获胜者存储在333号的位置，从而实现空间复用的目的，减少存储内存。

当然，在比赛中，还是采用临时数组的做法比较好，不容易出错且写的快。而且大部分情况下，都是对做法的时间做限制，而不是对空间做限制。

在胜者组只剩一人后，那这人就是冠军。观察我们的模拟过程，可以发现，进入败者组的时间越晚，淘汰的越晚。所以最后一个进入败者组的人，就是亚军。

但是季军就需要商酌一下了。在现实常规的比赛中，往往是第3与第4名争夺季军。但是由于该题的比赛策略有所不同，因此可能会出现在半决赛时，存在两种情况，即：三人半决赛、四人半决赛。

三人半决赛的话，那第三人就是默认季军了

而如果是四人半决赛的话，那么败者组两人就需要再pk一次，从而选出季军。

    def __ge__(self, other):

import java.util.*;

class Node implements Comparable<Node> {
    int id;     // 编号
    long value; // 值

    public Node(int id, long value) {
        this.id = id;
        this.value = value;
    }

    @Override
    public int compareTo(Node other) {
        // 重载比较方法，用于排序，按值从小到大排序，若值相等则按编号从小到大排序
        if (this.value == other.value) {
            return Integer.compare(this.id, other.id);
        }
        return Long.compare(this.value, other.value);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = 0;
        List<Node> win = new ArrayList<>();
        List<Node> lose = new ArrayList<>();

        // 读取输入值
        String[] input = scanner.nextLine().split(" ");
        n = input.length;

        for (int i = 0; i < n; i++) {
            win.add(new Node(i, Long.parseLong(input[i])));
        }

        boolean flag = false;

        while (win.size() > 1) {
            List<Node> tmpWin = new ArrayList<>();
            int tmp = win.size();
            for (int i = 0; i < tmp - 1; i += 2) {
                if (win.get(i).compareTo(win.get(i + 1)) >= 0) {
                    // 将败者添加到败者组列表
                    lose.add(win.get(i + 1));
                    // 将胜者移动到下一轮
                    tmpWin.add(win.get(i));
                } else {
                    lose.add(win.get(i));
                    tmpWin.add(win.get(i + 1));
                }
            }
            if ((tmp & 1) == 1) {
                // 处理奇数个选手的情况
                tmpWin.add(win.get(tmp - 1));
            }
            win = tmpWin;

            if (tmp == 4) {
                flag = true;
            }
        }

        // 打印结果
        System.out.print(win.get(0).id + " " + lose.get(lose.size() - 1).id + " ");
        if (!flag) {
            System.out.println(lose.get(lose.size() - 2).id); // 打印第三名选手（如果剩下三个选手）
        } else {
            // 确定第三名选手（如果剩下四个选手）
            System.out.println(lose.get(lose.size() - 2).compareTo(lose.get(lose.size() - 3)) >= 0
                    ? lose.get(lose.size() - 2).id : lose.get(lose.size() - 3).id);
        }
    }
}

type Node struct {

func (n Node) Less(other Node) bool {
	if n.value == other.value {
		return n.id > other.id
	}
	return n.value < other.value
}

# Js

function main() {
    const a = []; // 存储输入数据的数组
    let n = 0; // 输入数据的数量

    const readline = require('readline');
    const rl = readline.createInterface({
        input: process.stdin,
        output: process.stdout
    });

    // 逐行读取输入数据
    rl.on('line', (line) => {
        const values = line.split(' ');
        for (let i = 0; i < values.length; i++) {
            const x = parseInt(values[i]);
            a.push(x); // 将数据添加到数组中
            n++; // 数据数量加一
        }
    });

    rl.on('close', () => {
        const win = []; // 胜者组数组
        const lose = []; // 败者组数组
        let winNumber = n; // 胜者组中的选手数量
        let loseNumber = 0; // 败者组中的选手数量
        let flag = false; // 是否是四人半决赛

        // 将所有选手都加入胜者组
        for (let i = 0; i < n; ++i) {
            win[i] = { id: i, value: a[i] }; // 使用对象存储选手的id和值
        }

        // 比赛直到胜者组只剩下一个人
        while (winNumber > 1) {
            const tmp = winNumber;
            winNumber = 0;

            // 两两比较选手，胜者进入下一轮，败者进入败者组
            for (let i = 0; i < tmp - 1; i += 2) {
                if (
                    win[i].value >= win[i + 1].value ||
                    (win[i].value === win[i + 1].value && win[i].id < win[i + 1].id)
                ) {
                    lose[loseNumber++] = win[i + 1];
                    win[winNumber++] = win[i];
                } else {
                    lose[loseNumber++] = win[i];
                    win[winNumber++] = win[i + 1];
                }
            }

            // 处理奇数个选手的情况，最后一个选手直接进入下一轮
            if (tmp % 2 === 1)
                win[winNumber++] = win[tmp - 1];

            // 记录是否是四人半决赛
            if (tmp === 4)
                flag = true;
        }

        // 输出结果
        process.stdout.write(`${win[0].id}${lose[loseNumber - 1].id}`);

        if (!flag) {
            // 三人半决赛，输出第三名选手
            if (loseNumber >= 2) {
                if (loseNumber === 2)
                    process.stdout.write(`${lose[0].id}`);
                else
                    process.stdout.write(`${lose[loseNumber - 2].id}`);
            }
        } else {
            // 四人半决赛，败者组再战，输出第三名选手
            if (loseNumber >= 3) {
                if (lose[loseNumber - 2].value >= lose[loseNumber - 3].value)
                    process.stdout.write(`${lose[loseNumber - 2].id}`);
                else
                    process.stdout.write(`${lose[loseNumber - 3].id}`);
            }
        }

        process.stdout.write('\n'); // 输出换行符
    });

    rl.on('SIGINT', () => {
        rl.close();
    });
}

main();
