# 题目描述

塔子哥是一个数学爱好者，他对数字和分数充满了好奇心，最近他得到了一个分数 AB\frac{A}{B}​B​​A​​ 的分数，请你计算出 1−AB1 - \frac{A}{B}1−​B​​A​​ 的值。

# 输入描述

第—行给出两个整数 A,BA,BA,B，分别表示分数的分子和分母。

1≤A<B≤1031 \le A < B \le 10^31≤A<B≤10​3​​

# 输出描述

直接输出 1−AB1 - \frac{A}{B}1−​B​​A​​ 的值，无需化简，以分数的形式表示，分别输出分子和分母，用空格间隔。(输出必须满足分母为 BBB)

6 11

5 11

# 思路：模拟

输出b−ab-ab−a，bbb即可

# c++代码

#include <iostream>

using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    cout << b - a << ' ' << b << endl;
    return 0;
}

python代码

a, b = map(int, input().split())
print(b-a, b)

Java代码

import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A = sc.nextInt();
        int B = sc.nextInt();
        System.out.print(B - A);
        System.out.println(" " + B);
    }
}
